ğŸ¯ About the Project

Rental bikes improve urban mobility, but predicting demand is critical for maintaining a stable supply. This project uses historical bike rental, weather, and holiday data to forecast hourly bike demand in Seoul.

The dataset was sourced from the UCI Machine Learning Repository and described in studies by Sathishkumar V E, Jangwoo Park, and Yongyun Cho.
